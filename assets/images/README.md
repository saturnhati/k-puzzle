# K-Pop Idol Images

Place your K-pop idol images in this folder and update the image URLs in `src/renderer.ts`.

## How to add custom images:

1. Add your K-pop idol images to this folder (e.g., `idol1.jpg`, `idol2.jpg`, `idol3.jpg`)
2. Update the `idolImages` array in `src/renderer.ts` to use your local images:

```typescript
const idolImages = [
  '../assets/images/idol1.jpg',
  '../assets/images/idol2.jpg',
  '../assets/images/idol3.jpg'
];
```

## Image recommendations:
- Use square images (400x400 pixels or similar)
- JPG or PNG format
- High quality for better puzzle experience
